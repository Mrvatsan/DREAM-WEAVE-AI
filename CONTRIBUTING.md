# Contributing to DreamWeave AI

We welcome contributions! Please follow these guidelines:

1.  **Fork** the repository.
2.  Create a **Feature Branch** (`git checkout -b feature/NewMagic`).
3.  Commit your changes (`git commit -m 'feat: Add some Magic'`).
4.  Push to the branch (`git push origin feature/NewMagic`).
5.  Open a **Pull Request**.

## Coding Standards
- Use **ES6+** syntax.
- Follow **Tailwind CSS** utility patterns.
- Ensure **JSDoc** comments are present for new functions.
